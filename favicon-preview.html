A.S. Copy Favicon

How to Save as PNG:

  1. Right-click on the logo above
  2. Select "Save image as..." or "Copy image"
  3. Save as favicon.png

Or use an online converter:

  1. Go to svgtopng.com or cloudconvert.com
  2. Upload the favicon.svg file
  3. Set size to 512x512 or 1024x1024
  4. Download the PNG
